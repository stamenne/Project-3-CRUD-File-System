# OwlTech CRUD File System
## Overview
This short project acts as a file explorer system that allows users to perform CRUD (create, read, update, and delete) operations with files. 
The goal was to keep it simple and extremely user-friendly while still handling the basic functions of a file organization system.


## Main Operations
1. Create a File/Folder
2. Read a File/Folder
3. Update a File/Folder
4. Delete a File/Folder

## Additional Operations
1. Rename Files/Folders
2. Move files to a selected folder
3. Copy files to a selected folder

## Dependencies
1. Python 3.0+

## How To Run
Install Python 3.x and run the file from a terminal. All instructions for further operations are made clear with prompts.
